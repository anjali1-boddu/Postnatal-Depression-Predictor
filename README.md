# Postnatal-Depression-Predictor
🧠 Postpartum Depression Risk Prediction

A Machine Learning-based web application designed to predict the risk of Postpartum Depression (PPD) at an early stage using clinical and behavioral indicators.

🚀 Overview

This project leverages Random Forest Classification to provide early risk assessment for postpartum depression. The model is deployed using Flask as a REST API, enabling real-time predictions through a simple web interface.

🛠️ Tech Stack

Python

Scikit-learn

Pandas

NumPy

Flask

REST APIs

📊 Features

Early prediction of Postpartum Depression risk

Random Forest model for high accuracy and robustness

REST API integration for real-time predictions

Clean and lightweight Flask deployment

User-friendly web interface

⚙️ How It Works

User inputs relevant health and behavioral parameters.

Data is preprocessed using Pandas and NumPy.

The trained Random Forest model predicts risk level.

Results are displayed instantly via Flask API.
